== 프로젝트 설명 ==
기능1 : 개좌개설
기능2 : 입 금
기능3 : 출 금
기능4: 전체고객 잔액조회

== 프로그램 가정 ==
- 통장의 계좌는 중복을 허용하지 않는다
- 입금 및 출금 액은 무조건 0이상이다
- 고객의 정보는 계좌번호, 이름. 잔액 3가지만 저장한다
- 둘 이상의 고객 정보를 저장하기 위해 배열을 사용한다.
- 계좌번호는 정수 형태이다.


== 실행 예시 (계좌 개설과정 예시)==
1. 계좌 계설
2. 입 금
3. 출 금
4. 계좌정보 전체 출력
5. 프로그램 종료
선택 : 1

[계좌계설]
계좌ID : 115
이름 : 이우석
입금액 : 15000


== 실행 예시 (입금 예시)==
1. 계좌 계설
2. 입 금
3. 출 금
4. 계좌정보 전체 출력
5. 프로그램 종료
선택 : 2

[입 금]
계좌 ID : 115
입금액 : 70
입금완료

== 실행 예시 (정보 조회의 예시)==
1. 계좌 계설
2. 입 금
3. 출 금
4. 계좌정보 전체 출력
5. 프로그램 종료
선택 : 4

계좌 ID : 115
이 름 : 이우석
잔 액 : 15070
